# Coupon UI

A simple approach of coupon UI design. It's awesome UI to show the offer section. It's responsive.

## Uses 

The Coupon UI easy to use. Just you need to clone or download the github repo. After that you can copy [coupon.css](https://github.com/raishvaria/Coupon/blob/master/coupon.css) or [coupon.scss](https://github.com/raishvaria/Coupon/blob/master/coupon.scss)  file and coupon UI from [index.html](https://github.com/raishvaria/Coupon/blob/master/index.html).
